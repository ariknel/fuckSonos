REST API

All endpoints are available on the local network. Use these from Home Assistant or any HTTP client.

GET /api/speakers
List all discovered UPnP speakers
POST /api/stream/start
Begin streaming to selected rooms
POST /api/stream/stop
Stop streaming, send UPnP Stop
POST /api/volume
Set volume · body: {room, value}
GET /api/status
BT, WiFi, sync state as JSON
Factory reset
This will erase all Bluetooth pairings, WiFi credentials, and saved room configurations from NVS flash.
Phone disconnected
Bluetooth A2DP stream dropped. A Stop command has been sent to all active speakers.

When your phone reconnects, streaming will auto-resume to the same rooms. No action needed for brief disconnections.
